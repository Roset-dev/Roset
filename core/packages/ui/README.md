# <img src="../../../logo.png" width="32" height="32" align="center" /> @roset/ui

Roset's React component library. Build "Mission Control" interfaces with spatial design and dark-mode defaults.

## Usage

```bash
npm install @roset/ui
```

```tsx
import { Button, Card } from '@roset/ui';

export const App = () => (
  <Card variant="spatial">
    <Button intent="primary">Execute</Button>
  </Card>
);
```

[UI Documentation](https://docs.roset.dev/ui)
