# UI Package - Pre-built Library

This directory contains the **compiled** version of `@roset/ui` for distribution.

## Source Code

The source code for this package is **proprietary** and maintained separately.

## Usage

Install the pre-built package:

```bash
npm install @roset/ui
```

Use in your React application:

```typescript
import { Button, Card, Input } from '@roset/ui';

function App() {
  return (
    <Card>
      <Input placeholder="Enter text..." />
      <Button>Submit</Button>
    </Card>
  );
}
```

## What's Included

This package includes:
- Compiled JavaScript (ESM and CJS)
- TypeScript declarations (`.d.ts` files)
- Optimized for production use
