        Create expiring, revocable share links with fine-grained permissions. ACLs inherit through the directory tree, independent of underlying cloud IAM policies.
      </ChangeItem>
      <ChangeItem title="Audit Logging & Trash">
        A complete, append-only operation log for every action. Deleted items move to a virtual Trash with configurable retention and instant restore capabilities.
      </ChangeItem>
